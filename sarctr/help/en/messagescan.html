Message Scan

Message Scan allows user to scan messages, or transactions. It supports direct API call, Http request, IBM Websphere MQ or Java Message Service interface.

For example, user puts the messages into the input queue and system will pick it up and send to the filter engine to scan it; then load into Alert database for review. Please contact us for the setup and options.
