Sanction List

Sanction list manages the main entry of the sanction entity list. Click the Sanction List in the left navigation menu under List management, a new Sanction list Tab folder will created in the right panel. There are 3 sub-Tab folders, Search SDN List, Search result SDN List and Create New SDN List.

  • Search SDN List
  • Search Result SDN List
  • Update/Delete SDN List
  • Create New SDN List

a) Search SDN List

Search Criteria

entNum This is the unique number for the sanction list entry in the system.
Name Name of SDN.
Type Type of SDN, It can be Entity, Individual, Vessel or Others.
Program Sanctions program name.
Title Title of an individual.
Origin The source of this SDN, can be OFAC, OFSI, etc.
Remark Remarks on SDN
Address Stress address of SDN
City City of SDN
Country Country of SDN
Nationality Nationality of SDN
Citizenship Citizenship of SDN
DateOfBirth Date of Birth of Individual
PlaceOfBirth Place of Birth of Individual
AND and OR Select AND radio box if you want those criteria add together.
Select OR radio box if you want to get the result of either criteria meet

Functional buttons

  1. Find:
    click Find button to search.
  2. Clear:
    click Clear button to clear all the criteria.
  3. Complex Search:
    switch to Complex search mode.

b) Search Result SDN List

The top part is the table, shows the list of the sanction list that meet the search criteria. The bottom part is the details, Alts (Alternative name, or Alias) and GGs (Good Guys, or Exception) of the selected entry. Each entry can have 0 to many Alts (Alias) and GGs. (see Alias Name and Exception for details on Alts and GGs

c) Update/Delete SDN List

Double Click : the selected record in the table to modify this entry. A new Tab folder "Update/Delete SDN list" will be added, as show below.

Functional buttons

  1. Update:
    Make changes on the SDN list, and save and switch back to "Search result SDN list" Tab. Some of the fields in this form are read only, like entNum, created/modified user and date.
  2. Reset:
    reset the data to its initial value, discard all the changes.
  3. Cancel:
    Cancel this change and switch back to "Search result SDN list" Tab.
  4. Delete:
    Delete this entry; this will also delete the corresponding Alts and GGs if any. Then switch back to "Search result SDN list" Tab.

d) Create New SDN List:

Select the "Create New SDN List" Tab to create a new entry.

Field Name Descriptions.
entNum This is the unique number for the sanction list entry in the system. The value of entNum will be "LS" + sequence number. It is a read only field.
Name Name of SDN. This is a mandatory field.
Type Type of SDN, It can be Entity, Individual, Vessel or Others; This is a mandatory field.
Program Sanctions program name.
Title Title of an individual.
Origin The source of this SDN, It can be OFAC, OFSI, etc. ; This is a mandatory field
Remark Remarks on SDN
Address Stress address of SDN
City City of SDN
Country Country of SDN
Nationality Nationality of SDN
Citizenship Citizenship of SDN
DateOfBirth Date of Birth of Individual
PlaceOfBirth Place of Birth of Individual
CreatedByDate Date and Time when the SDN is created. It is a read only field.
CreatedByUser User who create this SDN. It is a read only field.
ModifiedByDate Last modified date and time; It is a read only field.
ModifiedByUser Last modified user; It is a read only field.

Functional buttons

  1. Create:
    Click the Create button to create a new SDN entry and switch to "Search result SDN List" Tab. Name, Type and Origin are mandatory fields, if either one of them are empty, an error will show.
  2. Clear:
    Click the Clear button to clear all the fields to empty.
  3. Cancel:
    Click the Cancel button to cancel operation and switch to "Search result SDN List" Tab.
