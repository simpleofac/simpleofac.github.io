Survey Category

All surveys are organized in survey category. Each category contains surveys and sub-categories. The top level category is root category. This root category is created by ATS with initial installation. It is editable but can not be deleted.

A list of surveys will show in the current category. To jump to another category, select from category drop-down box.

Functional buttons

  1. New sub-category:
    create a new category under current category
    • Category Name: Name of category
    • sort order: sort survey and sub-category by alphabetic, created date, or created date reversed
    • Description: Description of category

    After new sub-category is created, current category will be changed to this new category.

  2. Edit Category:
    Make changes on the category name, sort order and description
  3. Delete Category:
    delete current category. Category can not be deleted if there are any surveys or sub-categories. You must delete the surveys and sub-categories first.
  4. New Survey:
    create a new survey in current category
    • survey name: name of the survey
    • survey heading (title): which will display in the survey.
    • survey theme: select the theme of the survey, there are blue, green, yellow, pink, grey and white

Links

The links direct you to edit the selected survey. For detail, see survey section.

From the list of surveys in the current category, you can click the link on

  1. Survey name:
    go to survey dashboard of current survey.
  2. Edit:
    go to question list of current survey
  3. Send:
    go to survey invitation of current survey
  4. Preview:
    preview current survey
  5. Delete:
    delete current survey

Navigation bar

Show you current category and link to all its parent categories

Jump to another category:

There is a select drop-down box, which list all the categories within ATS. To jump to another category, simple select the category from the list. The category name ending with ** is the current category.
