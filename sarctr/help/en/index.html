Overview

SimpleOFAC SAR/CTR/E-Filing™ is built on the SimpleOfac OFAC/AML Suite platform. It uses the latest Web 2.0 (AJAX), providing user friendly interface, high performance and quality.

1 Manage CTR,SAR
Automatically generate CTRs, SARs if you are using our monitoring system, Simple AML suite. Easily pre-populate basic information, such as Filing Financial Institutions, Transmitter etc. Track progress from preparation to filing.
   
2 Search and Copy
Various search functions make it easy to find your CTR/SAR. Creating a new CTR or SAR in a few minutes with copy function.
   
3 Automatic Validation
Validate all the user input before it is saved and batch files for efile submission are generated. Batch efile has been fully tested in the BSA Efile test system. It will not be rejected by FINCEN due to format and/or data quality.
   
3 Workflow and Notification
It has easy to use, configurable workflow, built-in support for document management, email and audit log.
