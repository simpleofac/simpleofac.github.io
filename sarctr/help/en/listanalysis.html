List Analysis

List analysis gives users the ability to view the changes on the sanction list between day one and day two.

Click the "List Analysis" in the left navigation menu under List management, a new "List Analysis" Tab folder will created in the right panel. There are 2 sub-Tab folders, Search List Analysis, Search result List Analysis

  • Search List Analysis
  • Search Result List Analysis

a) Search List Analysis

Search Criteria

From Date The start date; it must be in mm/dd/yyyy HH:MM:SS or mm/dd/yyyy format. This is required.
To Date The end date; it must be in mm/dd/yyyy HH:MM:SS or mm/dd/yyyy format.
Leave this field empty will bring the latest data, the same as current date.
EntNum Link to SDN record identifier

Functional buttons

  1. Find:
    click Find button to search.
  2. Clear:
    click Clear button to clear all the criteria.

b) Search Result Sanction History

The left side, result list is the table, shows the list of the sanction list that has been changed (add/delete/update) during from-date to to-date. In the right top port, is the Details - Old, shows the select entry at From-Date point of time.
The right bottom part, Details - New, shows the selected entry at To-Date point of time.
If at From-Date, the select entry does not exist yet, name, type, program, etc fields will be empty.
If the select entry is deleted, The ****DELETE*** will be add in front of the name.
