Upload History

Every SDN list upload will create a record in the list upload history table. Like Sanction History, It can be used as audit purpose.

This function is used to search list upload history. Click the "Upload History" in the left navigation menu under List management, a new "Upload History" Tab folder will created in the right panel. There are 2 sub-Tab folders, Search Upload History and Search result Upload History.

  • Search Upload History
  • Search Result Upload History

a) Search Upload History

Search Criteria

Pub Date The date list published.
Load Date The date that this file is loaded.
Load By User who made the change
File Name The loaded file name.
AND and OR Select AND radio box if you want those criteria add together.
Select OR radio box if you want to get the result of either criteria meet

Functional buttons

  1. Find:
    click Find button to search.
  2. Clear:
    click Clear button to clear all the criteria.
  3. Complex Search:
    switch to Complex search mode.

b) Search Result Upload History

The top part is the table, shows the list of the upload history that meet the search criteria. The bottom part is the details of the selected entry. There is no create/update/delete function in Upload History
