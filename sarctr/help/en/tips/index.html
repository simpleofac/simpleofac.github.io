General Navigation Tips

 

  1. Select any menu item in the left side navigation menu will open a new Tab in the center panel, if the tab does not exist; otherwise, the existing Tab will be selected.
  2. Tables: select the header column of the table will sort the data on the column (if the column is sort-able). Right click the table header column will show a popup menu, allowing user to hide/show the columns
  3. Select a row (by click on the row) in the table will update the details panel in the south side of the page.
  4. If the table is updatable, double click on the select row will open a new Tab, allowing user to update/delete the record in the center panel, if the tab does not exist; otherwise, the existing Tab will be selected.
  5. When user is in the Create New Tab or Update/Delete Tab, other Tabs are disable. User has to click on Cancel, Create or Save button to switch to other Tab.
  6. Any changes(add/modify/delete) will also be recorded in the respective history table and retrieve via respective history function
  7. The search and search result can be in the same Tab or separate Tab. This is configurable.

    8. How does it work

    1 Scan messages
    Filter engine scans text/message(your customer data or transaction) against the sanction list, gives alert or no alert result. It supports free-text lookup or batch file scan.
    Contact us if you need to integrate with your system via Message service, Web service or/and direct API.
       
    2 Alert Management
    The alert result is saved into alert review system for review. It has build-in algorithm to detect duplicated text/message
       
    3 Reports
    You can generate the reports any time. We have message report, watch(sanction) list report.
    If you need other reports,please let us know.
