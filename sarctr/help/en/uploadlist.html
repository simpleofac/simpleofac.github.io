Upload List

This function allows user to load large volumes of data into List management. Currently it supports SDN XML file format. Other lists format can also be easily support, upon request.

Click the "Upload List" in the left navigation menu under List management, a new "Upload List" Tab folder will created in the right panel.

Click on Next button to go to file upload page

Click the Browse.. Button will open a window; then select a file to be uploaded. The file will be saved temporarily in the server, to avoid this upload file will override the file previous uploaded, user can check the "append timestamp" checkbox to append the current timestamp into file name.

Click Submit Button will upload the file into server, depends on the size of the size, It may take a couple minutes to upload.

Click OK to close this popup window; then click Next Button

Origin Set the origin for this list to load into. This is a mandatory field.
Origin must be at least 3 chars.
File Fame Preset from the previous page. This is the file just uploaded.

Functional buttons

  1. Generate Compare Report:
    click this button will load the file into a temporary database, which will be used in the next step to view the comparison report. A popup window will show the total number of records in the file and the number of added, deleted, updated and no changed records.

Click Next button will show the SDN that has been affected by this upload

If user is happy with the comparison report, Click Next to go to the upload form

Origin Set the origin for this list to load into. This is a mandatory field.
Origin must be at least 3 chars.
File Fame Preset from the previous page. This is the file just uploaded.

Functional buttons

  1. Load the File:
    click this button will load the file into list management SDN database. A popup window will show the total number of records in the file, and the number of added, deleted, updated and unchanged records
Please be note that user should not change the Origin and File Name in this page, otherwise, data loaded into the database will be not the same as we see the comparison report.
