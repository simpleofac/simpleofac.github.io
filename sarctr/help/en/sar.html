SAR

SAR manages the SAR list. Click the SAR in the left navigation menu under SAR & CTR; a new SAR Tab folder will appear in the right panel. There are 3 sub-Tab folders: Search SAR, Search result SAR, and Create New SAR.

  • Search SAR
  • Search Result SAR
  • Update SAR
  • Create New SAR

a) Search SAR

Search Criteria

SAR Id This is the system generated unique number for SAR entry in the system.
SAR Title Name of SAR, used to refer to a SAR.
DCN# Document Control number. It is optional
BSA Id Internal BSA Id, it is optional.
Org unit Code The org unit this SAR belongs to. It is used to control access
Assign To Assign to an individual
Each SAR has a status and can assign to an individual or a group.
Assign To Group Assign to a Group
Status Code Current status of the SAR
Assign Date The date when the SAR is assigned to
AND and OR Select AND radio box if you want those criteria add together.
Select OR radio box if you want to get the result of either criteria meet

Functional buttons

  1. Find:
    click Find button to search.
  2. Clear:
    click Clear button to clear all the criteria.

b) Search Result SAR

The top part is the table which shows the list of the SARs that meet the search criteria. The bottom part are the details. There are Details, Suspects(4A), SA Occurred FI(2B), Document, Email and Action Log Tabs,

In the Details Tab, there are SAR - General, Type of Filing, Filing FI(2A) and Suspicious Activity(3A), 3A Continued, Narrative(5A) sub Tabs
Each SAR can have 1 to many Suspects(4A) and SA Occurred FI(2B) records.

c) Update SAR

Double Click the selected record in the table to modify this entry. A new Tab folder "Update SAR" will be added, as show below.

Action Options

  1. SAR Action:
    Select an action to perform. The action allowed to perform is configured via the workflow along with the user permission.
  2. Action Reason:
    Select a pre-defined action reason in the dropdown menu.
  3. Comments:
    Add comments in the comments text area.

Functional buttons

  1. Submit:
    Depends on the action the user selected. Some actions will open a pop-up window or open a new Tab to allow the user to input more data.
  2. Reset:
    reset the data to its initial value, discard all the changes.
  3. Close:
    Discard this change (if any) and switch back to "Search result SAR" Tab.

d) Create New SAR:

Select the "Create New SAR" Tab to create a new entry. The mandatory fields in all tabs must be filled.
The easy way to create a new SAR is by copying from another One.

Field Name Descriptions.
SAR Id This is the system generated unique number for SAR entry in the system.
SAR Title Name of SAR, used to refer to a SAR.
Case Id The Sar is linked to the case Id if the SAR is created from a Case in the Case Management system.
DCN# Document Control number. It is optional
BSA Id Internal BSA Id, it is optional.
Assign To Assign to an individual
Each SAR has a status and can assign to an individual or a group.
Assign To Group Assign to a Group
Org unit Code The org unit this SAR belongs to. It is used to control access
SAR Explain Explain why we need to create this SAR
Action Reason Select a pre-defined action reason in the dropdown menu.
Comments Add comments on creating the SAR

Functional buttons

  1. Create:
    Click the Create button to create a new SAR and switch to "Search result SAR" Tab. Some fields are mandatory, an error pop-up window will show if the data does not pass the validation.
  2. Reset:
    Click the Reset button to reset all the fields to initial value.
  3. Close:
    Click the Close button to cancel operation and switch to "Search result SAR" Tab.
