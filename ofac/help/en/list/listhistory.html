Sanction List HistoryÂ¶

For every change (add/delete/modify) to the Sanction List entry, Alt. Name or Exceptions will create a record in the sanction history table. This not only serves as an audit purpose but it also can be used to restore the data to any [ast point in time.

This function is used to search sanction list history. Click the Sanction List History in the left navigation menu under List management and a new Sanction List History Tab folder will be created in the right panel.

There are 2 sub-Tab folders: Search Sanction List History and Search Result Sanction List History.

Note

There is no Create New or Update/Delete function on sanction list history.

Search Result Sanction List HistoryÂ¶

The top part is the table which shows the list of the Sanction List History that meet the search criteria. The bottom part is the details of the selected entry. There is no create/update/delete function in Sanction List History.
